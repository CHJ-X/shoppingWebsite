# 对接文档

## 登录

POST /login

### Request Body:

```
{
  "name": "姓名",
  "sex": "性别",
  "campus": "校区",
  "college": "学院",
  "grade": "年级",
  "tel": "手机",
  "period": "时间段"
}
```

### response Body:

```
{
  "status": 200,
  "msg": "登录成功",
  "data": null
}
```

### 错误

```
400：手机号错误或是其他参数错误
409：该手机号已报名
```
